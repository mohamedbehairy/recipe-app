A modern, responsive Recipe App built with **React**, **TailwindCSS**, and **React Router**.  
Browse meals by category, view details, and explore ingredients and area of origin.

---

## Features

- Browse **all meals** or filter by **categories**.
- View **meal details** with ingredients and instructions.
- **Responsive** layout for mobile, tablet, and desktop.
- **Sidebar** navigation with toggle button on mobile.
- Skeleton loading while fetching data from [TheMealDB API](https://www.themealdb.com/api.php).
- Fully functional **React Router** setup for dynamic routing.
